# Pie.Default 默认饼图
饼状图是 Echarts 组件中的重要图表，业务中的 UI 多种多样，为了尽可能的支持更多样式的饼图，对饼状图进行了深度拓展

<Preview>
  <Story name="Pie.Default">
    <div>
      <Pie.Default pieSource={[
        [
          {name: '饼1', value: 1},
          {name: '饼2', value: 2}
        ]
      ]}></Pie.Default>
      <Pie.Default
        seriesGap
        pieSource={[
          [
            {name: '饼1', value: 1},
            {name: '饼2', value: 2}
          ]
        ]}
      ></Pie.Default>
    </div>
  </Story>
</Preview>

```javascript
import { Echarts } from '@/components/common';
const { Pie } = Echarts;

const Test = () => {
  return (
    <div>
      <Pie.Default
        pieSource={[
          [
            { name: '饼1', value: 1 },
            { name: '饼2', value: 2 },
          ],
        ]}
      ></Pie.Default>
      <Pie.Default
        seriesGap
        pieSource={[
          [
            { name: '饼1', value: 1 },
            { name: '饼2', value: 2 },
          ],
        ]}
      ></Pie.Default>
    </div>
  );
};

export default Test;
```

### 参数

| 参数      | 说明                                                                                      | 类型         | 默认值 | 必选 |
| --------- | ----------------------------------------------------------------------------------------- | ------------ | ------ | ---- |
| pieSource | 参考 series 中的 data 属性                                                                | array[array] | 无     | 是   |
| seriesGap | 饼图数据间隔，当<code>seriesGap=true</code>时，legend 需要使用<code>Pie.Legend</code>组件 | boolean      | false  | 否   |
| grid      | 参考 echarts 官方文档                                                                     | string       | 无     | 否   |
| title     | 参考 echarts 官方文档                                                                     | object       | 无     | 否   |
| series    | 参考 echarts 官方文档                                                                     | object       | 无     | 否   |
| color     | 参考 echarts 官方文档                                                                     | object       | 无     | 否   |
| tooltip   | 参考 echarts 官方文档                                                                     | object       | 无     | 否   |
| legend    | 参考 echarts 官方文档                                                                     | object       | 无     | 否   |
| option    | 参考 echarts 官方文档，该属性会覆盖以上所有属性                                           | object       | 无     | 否   |

注意：默认主题自带默认样式，如果需要修改，则传入对应属性，代码中会对参数进行“合并”处理；
