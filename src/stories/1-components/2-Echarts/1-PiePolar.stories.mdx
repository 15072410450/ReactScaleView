| 参数    | 说明                               | 类型   | 默认值 | 必选 |
| ------- | ---------------------------------- | ------ | ------ | ---- |
| bgColor | 饼图数据背景色                     | string | 无     | 否   |
| 其他    | 参考 <code>Pie.Default</code> 组件 | object | 无     | 否   |

注意：Pie.Default的饼图默认取series[0]系列中的data作为数据展示，而Pie.Polar则取series多个系列的data作为展示；
